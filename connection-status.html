Why You Can't See API Results

If you're trying to test the API from a browser and not seeing results, it might be due to one of these reasons:

  1. CORS Issues: Your browser may be blocking cross-origin requests from localhost:3000 to localhost:5001.
  2. Server Running: Make sure your backend server is running on port 5001.
  3. Network Blocking: Some browsers or security software might block localhost connections.

How to Verify Connection

Method 1: Check Terminal Output

In your terminal where you ran npm run dev, you should see:

Server is running on port 5000
✅ MongoDB Atlas connected successfully

Method 2: Use a REST Client

Try using a tool like Postman or Insomnia to make a GET request to:

http://localhost:5000/api/health

Method 3: Use curl in Terminal

curl http://localhost:5000/api/health
